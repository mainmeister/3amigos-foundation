# The 3 Amigos Foundation

A fan tribute to the movie *Three Amigos!* featuring honorary membership, a certificate, and a collectible pin.

## Features
- **Honorary Membership**: Join the foundation and become an honorary Amigo.
- **Official Certificate**: Receive a beautiful (fictional) certificate of membership.
- **Collectible Pin**: Get an exclusive 3 Amigos pin.

## Development
This project is built with:
- HTML/CSS
- TypeScript

### Build
To compile the TypeScript source:
```bash
npm run build
```
